# マナタネ (manatane) プロトタイプ

## 概要

スマートフォン向けの学習入口サービス「マナタネ」のプロトタイプです（仕様書 v0.3 準拠）。

「マナタネ」は「学びの種」を意味する造語です。
英字表記は `manatane` です。

本サービスは、ユーザーのその日の目的・気分・興味に合わせて、学び始めるための入口を提案します。診断結果は固定的なタイプ判定ではなく、「今日の学び方の入口」を示す参考情報です。

## 実装範囲

- スマートフォン専用UI（12画面）
- 1分診断（クイズ / 占い風 / ミニゲームの3形式）
- ミニゲーム2種（スライドパズル4×4、迷路8×8）
- 4軸（A=関心の対象 / B=知ろうとする対象 / C=変えたい方向 / D=関わり方の深さ）の組み合わせによる81通りの診断結果
- 81名のガイドキャラクター（古典的人物。9ジャンル×9名）
- ハイブリッド方式によるガイド判定（行動傾向タグ＋軸属性スコアマッチング）
- おすすめルート表示
- localStorage による一時保存（履歴最大10件）

## 未実装

- 会員登録
- ログイン
- 決済
- DB保存
- 外部API連携
- 本格アフィリエイト計測

## 技術構成

HTML / CSS / JavaScript / JSON / localStorage のみ。
React、Vue、Next.js、Firebase、Supabase、Stripe、外部API は使用していません。

## 起動方法

`index.html` をブラウザで開いてください（ローカルファイルからの fetch 制限がある場合は、簡易HTTPサーバ経由でアクセスしてください。例: `python3 -m http.server 8000` のうえ `http://localhost:8000/`）。

GitHub Pages で公開可能です。

## 注意

このプロトタイプは診断結果を固定的な能力判定として扱いません。

結果は「今日の学び方の入口」を示す参考情報です。

ガイドの一言は本人の発言ではなく、考え方を学習向けに要約したものです。
