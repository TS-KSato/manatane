# マナタネ (manatane) プロトタイプ

## 概要

スマートフォン向けの学習入口サービス「マナタネ」のプロトタイプです。

「マナタネ」は「学びの種」を意味する造語です。
英字表記は `manatane` です。

## 実装範囲

- 1分診断
- クイズ診断
- 占い風診断
- ミニゲーム診断
- 診断結果
- ガイド表示
- おすすめルート表示

## 未実装

- 会員登録
- ログイン
- 決済
- DB保存
- 外部API連携

## 起動方法

`index.html` をブラウザで開いてください。

GitHub Pagesで公開可能です。

## 注意

このプロトタイプは診断結果を固定的な能力判定として扱いません。

結果は「今日の学び方の入口」を示す参考情報です。
